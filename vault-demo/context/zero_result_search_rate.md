# Zero-Result Search Rate

**Owner:** Product · **Источник данных:** Analytics

## Что это и зачем

Доля поисковых запросов, не давших ни одного результата. Каждый такой запрос, момент разочарования: игрок пришёл за конкретной игрой, а ему пустой экран. Плюс это бесплатные данные о спросе, что люди хотят, но не находят

## Формула — детали

Числитель: Searches with 0 results

Знаменатель: Total searches

## Ловушки

1) >10% = поиск нуждается в доработке. Каждый десятый запрос впустую, слишком много
2) Трекать сами запросы: это unmet demand, неудовлетворённый спрос. Данные для бизнес-решений
3) Часть пустых результатов ожидаема: эксклюзивы конкурентов, несуществующие игры, но это не должно быть основной массой

## Как улучшать

1) Fuzzy search: не ломаться на опечатках, «Starbursttt» должно находить «Starburst»
2) Synonym mapping: «рулетка» = «roulette», «блэкджек» = «blackjack 21», расширять поисковый индекс
3) Fallback results: вместо пустого экрана показать похожие или популярные игры, не отпускать игрока с пустыми руками
4) Использовать для каталогового планирования: если 500 раз ищут игру, которой у тебя нет, это сигнал добавить
