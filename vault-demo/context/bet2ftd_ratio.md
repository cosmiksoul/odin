# Bet2FTD Ratio

**Owner:** Product · **Источник данных:** BI

## Что это и зачем

Ставки до первого депозита (если есть демо-режим). Показывает вовлечённость в продукт ещё до того, как игрок внёс деньги

## Формула — детали

Числитель: Count bets before FTD

Знаменатель: —

## Ловушки

1) Применимо только для продуктов с бесплатной игрой. В большинстве iGaming = 0 по умолчанию
2) Долгая демо-игра без первого депозита = игрок получает удовольствие бесплатно и не видит причины платить
3) Некоторые рынки (Индия, Бразилия) показывают высокую вовлечённость в демо, но низкую конверсию в первый депозит

## Как улучшать

1) Оптимизировать демо-режим: хватит бесплатных кредитов на 10-15 минут игры
2) Триггеры для перехода на реальные деньги: «Ваш выигрыш мог быть реальным»
3) Бесшовный переход из демо в реальную игру: один клик «Играть на реальные»
