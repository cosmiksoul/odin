# ARPU

**Owner:** Product · **Источник данных:** BI

## Что это и зачем

Средний доход на пользователя (Average Revenue Per User): основа для расчёта LTV и оценки качества когорт. Важно: считать по NGR, не по GGR, иначе цифра будет завышенной

## Формула — детали

Числитель: GGR или NGR

Знаменатель: Active Users (MAU)

## Ловушки

1) ARPU без сегментации бесполезен: 1 кит с ARPU ~$10 000 и 999 игроков с ARPU ~$5 дают средний ARPU ~$15, но это не описывает ни одного реального игрока
2) Всегда смотреть медиану рядом со средним. Если медиана намного меньше среднего, экономика держится на китах
3) ARPU по GGR и ARPU по NGR могут расходиться при росте бонусов: считать по NGR

## Как улучшать

1) Растить частоту депозитов через CRM-триггеры при низком балансе
2) Увеличивать средний депозит через предложенные суммы и бонусы за пополнение
3) Оптимизировать набор игр для увеличения времени на сайте и раундов за сессию
4) Кросс-продажи для увеличения доли кошелька (больше продуктов, больше тратит)
