# Latency (key flows)

**Owner:** Tech · **Источник данных:** Monitoring

## Что это и зачем

Время отклика ключевых потоков: депозит, ставка, запуск игры, вывод. Медленная платформа = брошенные депозиты и раздражённые игроки. Особенно критично для лайв-беттинга, где секунды решают.

## Формула — детали

Числитель: P50/P95/P99 response time (ms)

Знаменатель: Per flow (cashier, lobby, bet placement)

## Ловушки

1) Мониторьте P95+, не среднее, среднее скрывает проблемы «хвоста». 2) Целевые ориентиры: страница <2 секунды, API <200мс, запуск игры <3 секунды. 3) Мобильный трафик в развивающихся странах, latency в 2–3 раза выше, учитывайте это.

## Как улучшать

1) CDN для статического контента, картинки и скрипты должны летать. 2) Оптимизация базы данных, медленные запросы тормозят всё. 3) API optimization, минимизация payload, кеширование. 4) Гео-распределённая инфраструктура, серверы ближе к игрокам.
