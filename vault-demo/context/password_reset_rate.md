# Password Reset Rate

**Owner:** Tech / Security · **Источник данных:** Auth

## Что это и зачем

Доля пользователей, запрашивающих сброс пароля. Это индикатор трения при входе: чем больше сбросов, тем сложнее игрокам попасть в свой аккаунт. А каждый сброс, это шанс потерять игрока по дороге.

## Формула — детали

Числитель: Password resets

Знаменатель: Active users

## Ловушки

1) Больше 5% MAU = серьёзное трение, ищите способ убрать пароли вообще. 2) Высокий сброс + низкий возврат = игроки уходят во время процесса восстановления. 3) Может быть вектором социальной инженерии, мониторьте подозрительные паттерны.

## Как улучшать

1) Уменьшайте потребность: биометрия, magic link, SMS OTP, пароль вообще не нужен. 2) Простой и быстрый процесс сброса, оптимизированный под мобильные. 3) Поддержка password manager, автозаполнение снижает забытые пароли. 4) После сброса, редирект к изначальному действию, а не на главную.
