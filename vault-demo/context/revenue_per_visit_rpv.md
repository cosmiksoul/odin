# Revenue per Visit (RPV)

**Owner:** Marketing · **Источник данных:** BI

## Что это и зачем

Чистый доход (NGR) на один визит сайта. Связывает маркетинговый трафик с монетизацией, полезен для закупки рекламы и оценки посадочных страниц

## Формула — детали

Числитель: NGR

Знаменатель: Total visits (sessions)

## Ловушки

1) RPV кратно отличается по источникам: органический RPV может быть в 10 раз выше платного. Считать раздельно
2) RPV за день или неделю нестабилен: считать за 30+ дней для корректного среднего
3) RPV не учитывает LTV: посетитель может не конвертироваться сегодня, но вернуться через неделю

## Как улучшать

1) Растить конверсию на каждом шаге воронки: клик → регистрация → первый депозит
2) Увеличивать средний первый депозит через UX и предложенные суммы
3) Оптимизировать посадочные страницы под намерение (одна для казино, другая для спорта)
4) Снижать показатель отказов через скорость загрузки и релевантность контента
