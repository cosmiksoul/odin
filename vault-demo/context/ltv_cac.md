# LTV:CAC

**Owner:** Marketing · **Источник данных:** BI

## Что это и зачем

Соотношение пожизненной ценности к стоимости привлечения, эффективность инвестиций в привлечение. Ориентир: LTV:CAC больше 3x для устойчивого бизнеса. Меньше, бизнес не зарабатывает на привлечённых игроках

## Формула — детали

Числитель: LTV

Знаменатель: CAC

## Ловушки

1) LTV:CAC больше 5x может означать недоинвестирование в рост: есть запас для более агрессивного привлечения
2) LTV:CAC по общему трафику скрывает убыточные каналы: считать по каждому источнику отдельно
3) CAC через партнёров (CPA) и через платную рекламу, вещи разные: партнёрский включает только стоимость первого депозита, платный, все показы

## Как улучшать

1) Растить LTV: удержание, оптимизация дохода на игрока, снижение расходов на бонусы
2) Снижать CAC: оптимизация конверсии воронки, рост доли органики, пересмотр условий с партнёрами
3) Отключать каналы с LTV:CAC ниже 1.5x, они убыточные
4) Перенаправлять бюджет на каналы с доказанным LTV:CAC выше 3x
