# Deposit Limit Set Rate

**Owner:** Compliance · **Источник данных:** RG System

## Что это и зачем

Доля игроков, установивших лимиты на депозиты. Проактивная RG-мера: игрок сам ставит себе границу. Регуляторы всё чаще требуют этот показатель, и он влияет на продление лицензии.

## Формула — детали

Числитель: Users with deposit limits

Знаменатель: Active users

## Ловушки

1) Нормы сильно различаются: UK 5–10% добровольно, Швеция 30%+ (регуляторное давление). 2) Лимит в $99,999, это галочка, а не реальная RG-мера. Смотрите на адекватность сумм. 3) Set rate без usage rate, неполная картина: установили и забыли ≠ используют.

## Как улучшать

1) Подсказка при регистрации: «Хотите установить лимит?», мягкий nudge. 2) Дефолтные лимиты, пусть будут по умолчанию, игрок может поднять. 3) Позитивная подача: «Контролируй свою игру», не «Ограничь себя». 4) Предложения на основе дохода, помогают установить адекватный лимит.
