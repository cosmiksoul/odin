# Stickiness

**Owner:** Product · **Источник данных:** BI

## Что это и зачем

DAU/MAU ratio. Показывает, какая доля месячной базы приходит ежедневно. Proxy для habit strength (сила привычки условно)

## Формула — детали

Числитель: DAU

Знаменатель: MAU

## Ловушки

1) Stickiness (или липоксть, привычка) в iGaming ниже чем social apps: 15-25% = норма для casino, 10-15% для sports (event-driven)
2) Слишком высокая stickiness (>40%) может быть RG-сигналом: обязательно cross-check с responsible gambling metrics
3) Stickiness искажается дейли логин бонусами: игрок может логиниться для бонуса, не играя

## Как улучшать

1) Daily rewards и login bonuses
2) Лайв-события: стримы, турниры, daily джекпоты
3) Push-уведомления с персонализированным контентом
4) Streak mechanics: 'не прерывай серию дней' (берем со всем фритуплей игрушек)
