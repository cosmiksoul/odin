# PAU (Real)

**Owner:** Product · **Источник данных:** Game Aggregator

## Что это и зачем

Игроки с реальными (не бонусными) ставками. Настоящий показатель активной базы

## Формула — детали

Числитель: Unique users with ≥1 real money bet

Знаменатель: —

## Ловушки

1) Если PAU(Real) << PAU, база живёт на бонусах. При сокращении бонусов активность упадёт
2) PAU(Real) / PAU ratio показывает 'здоровье': цель >70%
3) Бонусные игроки, не конвертирующиеся в реальных, имеют негативных LTV (затраты на бонусы > ревеньюшка)

## Как улучшать

1) Конверсия bonus=>real play: после отыгрыша бонуса игрок должен продолжить на свои
2) Снижение bonus dependency: постепенное уменьшение бонусов для активных реальных игроков
3) Product value без бонусов: кешбэк, поинты лояльности, VIP привелегии
4) Реальные деньги для онбординга: первая ставка на реальные деньги с минимальным риском
