# Limit Usage Rate

**Owner:** Compliance · **Источник данных:** RG System

## Что это и зачем

Доля игроков, которые реально достигают своих лимитов. Показывает, работают ли лимиты как инструмент контроля или это просто формальность. Если лимит стоит, но никогда не срабатывает, он бесполезен.

## Формула — детали

Числитель: Users with any limit set

Знаменатель: Active users

## Ловушки

1) Больше 50% регулярно упираются в лимит, либо лимиты слишком высокие, либо нужно усиливать RG. 2) Систематическое достижение + запросы на повышение = at-risk игрок, нужна интервенция. 3) Без контекста времени сессии картина неполная.

## Как улучшать

1) Помогите игрокам установить осмысленные лимиты, не заоблачные суммы. 2) Уведомление при 80%, «Вы потратили 80% лимита» помогает осознать. 3) Период охлаждения после достижения лимита, не давайте сразу поднять. 4) Снижение лимита, мгновенно, повышение, с задержкой.
