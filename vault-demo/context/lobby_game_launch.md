# Lobby => Game Launch

**Owner:** Product · **Источник данных:** Analytics

## Что это и зачем

Конверсия из лобби в запуск игры. Показывает, насколько эффективно игрок находит интересную игру. Лобби, это витрина, и она должна продавать

## Формула — детали

Числитель: Game launches

Знаменатель: Lobby page views

## Ловушки

1) Высокая конверсия при низком покрытии каталога = игроки не исследуют, механизм открытий сломан
2) Конверсия для новых и вернувшихся игроков различается: новым нужна навигация, вернувшиеся идут в избранное
3) Глубокая прокрутка без запуска = игрок не находит ничего интересного. Пора проверить каталог

## Как улучшать

1) Персонализация лобби: «Рекомендовано для вас», «Популярное», «Новое», разные секции
2) A/B-тесты раскладки: сетка vs список, количество видимых игр, размер превьюшек
3) Поиск и фильтры по провайдеру, типу, волатильности
4) Секция «Последние» и «Избранное» для вернувшихся игроков
